# 🧮 C# Calculator

C#을 공부하면서 만든 간단한 계산기 프로그램입니다.

## 소개
이 프로젝트는 C# 기본 문법과 로직을 연습하기 위해 제작한 콘솔 기반 계산기입니다.  
사칙연산을 중심으로 기본적인 계산 기능을 구현했습니다.

## 기술 스택
- C#
- .NET

## 주요 기능
- ➕ 덧셈
- ➖ 뺄셈
- ✖️ 곱셈
- ➗ 나눗셈
- 사용자 입력 기반 계산
