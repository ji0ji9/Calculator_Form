# C# Calculator

C#을 공부하면서 Windows Form 기초를 익히기 위해 만든 계산기 프로그램입니다.<br><br>


## 소개
이 프로젝트는 C#의 기본 문법과 WinForms UI 구성, 그리고 간단한 로직 구현을 연습하기 위해 제작했습니다.  
아직 구현 과정에서 미흡한 부분이나 오류가 일부 남아 있지만, 학습 과정에서의 기록이라는 점에 의미를 두고 있습니다.

기본 계산 기능과 함께 진법 변환(HEX, DEC, OCT, BIN) 기능을 구현해보며 이벤트 처리와 데이터 변환 로직을 연습했습니다.

## 기술 스택
- C#
- .NET (Windows Forms)<br><br>


## 주요 기능
- ➕ 덧셈
- ➖ 뺄셈
- ✖️ 곱셈
- ➗ 나눗셈
- 🔢 진법 변환 (HEX / DEC / OCT / BIN)
- ⌨️ 버튼 입력 기반 계산<br><br>

- 예외 처리 부족 (잘못된 입력 처리 미흡)
- 코드 구조 리팩토링 필요<br><br>
